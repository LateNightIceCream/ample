# ample
custom audio amplifier with line and s/pdif inputs

## Features
| category | features                                                                       |
|----------|--------------------------------------------------------------------------------|
| output   | 15 W speaker output, headphone output                                        |
| input    | optical S/PDIF (Toslink connector), line-level analog audio (RCA connector)  |
| control  | GPIO header                                                                  |
| supply   | 24 VDC                                                                       |
